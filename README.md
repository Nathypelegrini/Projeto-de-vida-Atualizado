
# Atividade Prática: Criando e Visualizando Nosso Projeto de Objetivos
Prezados alunos,
Na aula de hoje, daremos início à construção da nossa página personalizada de objetivos do ano. Para garantir que o código de vocês fique salvo com segurança e que vocês consigam visualizar o resultado final (inclusive em casa), seguiremos um roteiro dividido em quatro etapas principais.
Por favor, sigam as instruções detalhadas abaixo com muita atenção à nomenclatura dos arquivos e aos comandos de cópia.
## 📋 Passo a Passo da Atividade
### Passo 1: Criando um Novo Repositório no GitHub
O GitHub será o local onde o código de vocês ficará armazenado publicamente de forma profissional.
 1. Acessem o site do **GitHub** e façam o login.
 2. No canto superior direito, cliquem no ícone **"+"** e selecionem **"New repository"** (Novo repositório).
 3. No campo **Repository name**, definam o nome exatamente como: meus-objetivos-do-ano.
 4. Certifiquem-se de que a opção **Public** está marcada.
 5. Selecionem a caixa **"Add a README file"**.
 6. Finalizem clicando no botão verde **"Create repository"**.
### Passo 2: Criando a Estrutura do Projeto
Agora, criaremos as pastas virtuais (arquivos) que receberão as linguagens da web.
 1. Dentro do seu repositório, cliquem em **"Add file"** e depois em **"Create new file"**.
 2. No campo de nome do arquivo, digitem exatamente: **index.html**
 3. Repitam esse processo para criar mais dois arquivos com os seguintes nomes precisos:
   * **style.css**
   * **main.js**
 4. **Muito importante:** Toda vez que criarem um arquivo ou colarem algo nele, vocês devem rolar a página e clicar no botão verde **"Commit changes..."** (Confirmar alterações) para salvar o arquivo de forma definitiva no GitHub.
### Passo 3: Copiando os Códigos do Google Classroom
A estrutura inicial foi disponibilizada pela professora no ambiente virtual.
 1. Acessem o link da atividade no **Google Classroom**.
 2. Abram o arquivo de código correspondente (HTML, CSS ou JS).
 3. Utilizem o atalho **Ctrl + A** para selecionar todo o texto ou cliquem no botão **"Copiar"** (ícone de prancheta).
 4. Voltem ao seu GitHub, entrem no arquivo equivalente, cliquem no ícone de lápis para editar e utilizem o comando **Ctrl + V** para colar o código. Não se esqueçam de realizar o *Commit*.
### Passo 4: Visualização no CodePen (Ideal para Praticar em Casa)
Para que vocês consigam ver o layout, os botões e o contador funcionando em tempo real, utilizaremos uma ferramenta online de testes chamada CodePen.
 1. Acessem o site do CodePen.
 2. Vocês encontrarão três campos principais na tela: um para **HTML**, um para **CSS** e um para **JS**.
 3. Utilizem o **Ctrl + V** para colar cada código copiado do Classroom em seu respectivo campo dentro do CodePen.
 4. A visualização do projeto (a interface da página) aparecerá instantaneamente na metade inferior da tela.
 5. Cliquem em **"Save"** para guardar o progresso. Essa ferramenta é excelente para que vocês continuem editando e testando os seus objetivos diretamente de casa.
> 📌 **Lembrete da Professora:** Atenção redobrada à nomenclatura dos arquivos no GitHub. Letras maiúsculas ou extensões erradas (como esquecer o ponto ou trocar as letras) farão com que o navegador não consiga ler o projeto corretamente. Bom trabalho a todos!
